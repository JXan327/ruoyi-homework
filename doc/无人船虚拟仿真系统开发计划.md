# 无人船虚拟仿真系统开发计划

## 1. 项目概述

本项目是基于若依（RuoYi）工程开发的无人船虚拟仿真系统，旨在实现无人船的虚拟仿真、监控和管理功能。系统采用前后端分离架构，前端使用Vue.js，后端使用Spring Boot，数据库使用MySQL。

## 2. 现有功能分析

### 2.1 项目结构

- **ruoyi-admin**：后端管理系统
- **ruoyi-common**：公共工具类
- **ruoyi-framework**：框架核心
- **ruoyi-generator**：代码生成器
- **ruoyi-quartz**：定时任务
- **ruoyi-system**：系统管理模块
- **ruoyi-ui**：前端界面

### 2.2 已实现功能

- 无人机场信息管理菜单配置
- 基础权限控制（查询、新增、修改、删除、导出）

### 2.3 数据库设计

已配置无人机场信息管理的菜单和权限数据。
